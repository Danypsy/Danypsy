## 🦅 FALCON Analysis Framework

This repository includes the **FALCON** (Futurism & Advanced Logic CONtext Framework) skill for Claude AI - a comprehensive analytical framework for deep strategic analysis of content.

### What is FALCON?

FALCON is a rigorous 10-module analytical framework designed for exhaustive analysis of content related to:
- AI and technological trends
- Futurism and speculative scenarios
- Complex systems thinking
- Epistemology and knowledge claims
- Technological philosophy
- Narrative analysis

### Features

- **10 Comprehensive Modules:** From thesis identification to meta-analysis
- **Methodological Rigor:** 5-15 paragraphs minimum per module
- **Analytical Neutrality:** Objective structural analysis
- **Epistemological Precision:** Clear distinction between fact and speculation
- **Alternative Scenarios:** Multiple plausible futures consideration
- **Meta-Analysis:** Evaluates originality vs. recycled ideas

### Usage

The FALCON skill is located in `.claude/skills/falcon-analysis.md` and can be invoked when working with Claude Code to perform deep analysis of any content.

See [.claude/skills/README.md](.claude/skills/README.md) for detailed usage instructions.
